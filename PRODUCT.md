# Product

## Register

product

## Users

Solo user tracking their personal gold coin and bar portfolio. Uses the tool on a desktop or laptop browser at home, checking holdings periodically rather than in real-time trading sessions. Values clarity and confidence in their investment data.

## Product Purpose

Track physical gold holdings (coins and bars) with cost basis, weight, purity, and real-time spot prices. Calculate portfolio P&L and maintain a clean record of acquisitions. All data stored locally in the browser — no backend, no accounts.

## Brand Personality

Precise, calm, confident. Feels like a well-crafted personal tool — not a corporate dashboard, not a trading terminal. Clean and personal, where data is presented with restraint and clarity. The interface should feel like a trusted companion for a serious collector.

## Anti-references

No SaaS clichés (hero metrics with gradient cards, hero-metric templates, identical card grids). No cream/beige/parchment warm-neutral defaults. No clutter or dense financial terminals. No generic shadcn/ui blue that feels like a starter template.

## Design Principles

- **Data first, decoration last**: Every visual element must earn its place by improving data comprehension.
- **Gold is the accent**: Color should reflect the domain — gold/warm tones as the primary accent, not an afterthought.
- **Restraint over density**: Present more information by reducing visual noise, not by cramming.
- **Dual elegance**: Both light and dark modes should feel intentional and on-brand, not like inverted defaults.

## Accessibility & Inclusion

WCAG 2.1 AA minimum. P&L indicators use both color and text — never color alone. All interactive elements have visible focus states. Table data is readable at standard sizes with proper contrast ratios.
